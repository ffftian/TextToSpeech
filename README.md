# TextToSpeech


TextToSpeech是附属于TRPGLogline的文字音频转换工具。
用这个软件进行声音的配音/录制来保证可以被TRPGLogline正常读取。
本教程的版本基于1.1版本。

# 目录
- [导出QQ记录](#导出QQ记录)
- [生成语音设置](#软件生成配置 )
# 导出QQ记录
点击所在日志的QQ记录，点击![消息管理](docs/1.png)选择想要导出的群组![消息导出](docs/2.png)   
保存类型设置为txt文件,您可以自由删除不必要的对话，这样就完成了文件准备工作。
